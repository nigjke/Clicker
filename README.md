# Сlicker

Программа для Windows, позволяющая добавлять точки для кликов мышью и запускать клики по ним с задержкой.

## Функционал

- **F8**: 
  - До F9 — добавить точку в список и клик по ней.  
  - После F9 — клик по текущему курсору + всем сохранённым точкам.  
- **F9**: завершить режим добавления точек.  
- **F10**: очистить все сохранённые точки.  

## Настройки

При запуске программы можно указать задержку между кликами (в миллисекундах).

## Архитектура

- `MouseClicker` — хранение точек и клики.  
- `HotKeyManager` — регистрация и обработка глобальных горячих клавиш.  
- `Program` — точка входа, ввод настроек.

## Пример использования

1. Запустите программу.
2. Введите задержку между кликами.
3. Наводите курсор и нажимайте F8, чтобы добавлять точки.
4. Нажмите F9, чтобы завершить добавление.
5. Теперь при нажатии F8 клик будет происходить сначала по текущему курсору, затем по всем сохранённым точкам.
6. F10 очистит все точки для нового цикла.

## Требования

- Windows OS  
- .NET Framework 4.7+ или .NET 6+
